霸总小说好看吗？好看

发表于 2021-02-23

进晚本来想岛建的，但是又突然不想了(?)于是来回忆并随便讲讲我至今看过的网络小说，都是女频文。感觉给每一篇开个阅读笔记似乎不合适，因为我全部都是跳着看的，而且除了甜一下爽一下也没什么更深刻的感想了。

霸道总裁好嗑吗？好嗑。

我向来都认为人就应该雅俗共赏，看过不好的才能分辨什么是好的，就像我的播放列表里从古琴独奏到西洋交响乐再到日式炸厕所音乐应有尽有，除了rap和爵士乐我是真的欣赏不来。当然这里没有网文或者哪种音乐类型不好的意思。有些网文即使受到受众人群的影响剧情发展需要向阅读量低头但真的还是挺有趣的，日式炸厕所音乐也非常带感，不然我也不会看不会听是不是。

最开始看网文是因为看到了一篇微博推荐，是的我那个时候还在用微博。推荐了一本娱乐圈霸总文，夜盛霆x沈安然，被开头吸引于是去搜了全文。之后觉得非常上头于是四处搜罗甜美小情侣嗑，口味限定在现代、不带系统的重生，娱乐圈除了最开始那篇也不怎么喜欢看了。后来开始看那种80多章的小甜文，还看了几篇“悬爱”(说实话觉得这个定义蛮好玩的)，之后又去看军婚和穿书，还有和冥王结婚(。)女频的悬疑恐怖基本就是阴婚到冥王这样子，换皮霸总。

（前段时间冥婚闹得非常凶，虽然那个时候我已经有很长一段时间没看网文了但是依旧情不自禁想到女频的悬疑恐怖榜。突然觉得好苦啊，好苦啊。虽然我不太信阴曹地府，就算有我更不相信活人能用阳间钱操控阴间事。但是冥婚这件事存在的本身却能反应一些观念和寄托——因为死后的命运将不受自己控制，所以只能通过文字幻想一下自己被配到了冥王——这么想真的太苦涩了。）

后来就没再找到什么我觉得还挺有意思能让我看下去的网文了，于是又缩回微信阅读薅羊毛读出版书了。希望有新的小情侣给我嗑~

因为网文的标题实在是太复杂了，并且都差不多，而且还会因为各种原因改名，所以我全部用男女主名字罗列，作者都是网名也不写了网上一搜就有。大体按照阅读顺序排列。

只是随便看了几章的那种就不列了。

+

  1. 夜盛霆 x 沈安然
    入坑作，看的时候二人刚生小女孩，之后就没追了。还不错，唯一能看得下去的娱乐圈浓度较高的作品。
  2. 慕非池 x 云曦
    又一篇微博推荐，挺好看的。看的时候未完结，更新频率不高所以没追完。
  3. 墨景深 x 季暖
    我目前最喜欢的小情侣，可惜据说副cp抄袭。一般都对副cp无兴趣我没看，说是抄袭了原耽。
    嘻嘻对原耽充满偏见的我寻思可能顶多算狗咬狗。
    （但是我真的超喜欢这对啊艹真实强强作品描写社会的氛围很棒白莲花也很持久(?)车也香的不行，人间不值得了）
  4. 司夜寒 x 叶绾绾
    马甲太多了跳着看没看懂，男女主都挺纸片的。但是前期意外上头。
  5. 陆霆骁 x 宁夕
    还行，人物纸片程度依然高出(我看到的)霸总文平均水平，作者风格吧可能。但是这对挺甜。
  6. 厉爵风 x 顾小艾
    宁就是割肾虐文开山鼻祖？
  7. 薄靳言 x 简瑶
    看到作者写完这本后还跟进了小剧场，结果在2中某位在小剧场里孩子都有了的角色死了，盲猜后来才构思的2。
    但是2挺难看的，就很狗血，我是真的喜欢1的这一对。
  8. 韩沉 x 苏眠
    挺狗血的……
  9. 厉致诚 x 林浅
    精简霸总文，车速较高，还不错啦。
  10. 季白 x 许诩
    没看完，作者很喜欢写婆婆阻碍婚事的剧情。
  11. 言朔 x 甄爱
    据说作者融梗？不评价了。看的时候还感叹过作者好博学。我倒是觉得哥哥团和《美人为馅》的那个组织有异曲同工之妙……
  12. 傅维珩 x 苏莞
    你妈的，纯爱战士高呼好看。虽然我一直拉琴但是让Kerishya成为小提琴x大提琴的想法还是来源于此。
  13. 慕承和 x 薛桐
    这对给予了我搞师生oc的灵感233333不过后来没搞下去，设定倒是还记得，有兴致就在博客里讲一讲。
  14. 顾魏 x 我
    忘记女主名字了……现实向，像日记。看完之后一直在找医生男主的文章。
  15. 何苏叶 x 沈惜凡
    男主是中医，忘记讲了什么了……只记得看的时候挺喜欢的。
  16. 陆淮 x 叶楚
    似乎是我唯一看到结局的非现代背景文。舍友推荐的，男女主很好嗑，但是最终boss排场没撑住，阴谋太掉价了。
    希望各位女频小作家写不来大阴谋就别硬写，被白莲花陷害陷害也不是不好看……
  17. 顾沉 x 简桑榆
    喜欢这对的相处模式，但是没追完不知道认亲后发展成什么样了。
  18. 权靖城 x 萧岚夜
    可以说是我觉得墨景深 x 季暖后第二好看的文了。
  19. 慕煜尘 x 席夏夜
    上文作者的早期作品吧，文笔太尬了汗。少数女配浪子回头的作品，看到热评猜测作者估计因此被读者骂死了哈哈哈哈。（脱离评论区的无情刷文机器）
  20. 夜司溟 x 陈秀秀
    很像男孩子写的？从一开始就在追，当时面临腰斩。这种发展在女频肯定很不吃香啊哈哈哈哈今天想起来还去看了一眼，因为我还挺喜欢的，结果800多章了二人只亲了一次(还是迫不得已)，还没睡过草。
    在一众冥王宠妻里能吃香就怪了，去男频不知道能不能挣扎一下。

想了想还是放在书籍阅读分类下了。我寻思我之后还能盘点一波恶役大小姐少女漫画(手动狗头)
上面有几篇后来剧改了吧，估计书评区已经变成追星刷屏专区了，嘻嘻。
说起来我扫荡完大妈之家我能找到的恶役大小姐漫画后卡塔莉娜也动画化了。我明白了，我看啥啥火(doge)
